# CZElauncher   CSGO僵尸逃跑社区服登录器
=========================================

*.国服黄页和X社区一键切换

*.每日签到礼包 

*.每日摇摇乐(摇摇乐可提前获知奖励) 

*.Steam帐号切换 

*.地图清理 

*.指令查询绑定 

*.商城皮肤预览 

*.服务器搜索(支持多选搜索)PS:现在可以多选搜索服务器了. 例如->僵尸,KZ 这样用 [,]分隔开 

*.新手礼包获取

*.地图识别检查下载(目前支持大多数常玩模式啦) 

*.皮肤主题切换 

*.背景图片自定义更换 

*.预热下载,显示游玩服务器 

*.地图中英文名切换显示 

*.查看服务器玩家
